# Виджет банковских операций

## Описание проекта

Виджет банковских операций - это учебный проект по Python.
Проект содержит код для разработки виджета банковских операций клиента.
Он маскирует номера карты или счёта клиента, сортирует операции по дате и фильтрует по выбранному статусу.

## Установка

1. Клонируёте репозиторий:
```
git clone https://github.com/Kirill-Leschinskiy/bank.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```

## Использование

1. Откройте приложение банка
2. Зайдите в личный кабинет

## Тестирование

Наш проект покрыт юнит-тестами Pytest. Для их запуска выполните команду:
```
poetry run pytest
```
